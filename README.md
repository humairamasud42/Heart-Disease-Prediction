📌 Heart Disease Prediction

🎯 Objective

To build a machine learning model that predicts whether a person is at risk of heart disease using medical health data.

📊 Dataset Used

* Heart Disease UCI Dataset
* Source: Kaggle

✒️ Features include:

* Age
* Sex
* Cholesterol
* Blood Pressure
* Chest Pain Type
* Heart Rate
* ECG Results

🧩 Target:

* Presence or absence of heart disease

🧠 Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

📈 Methods & Evaluation Used

✔ Data Cleaning

✔ Exploratory Data Analysis (EDA)

✔ Logistic Regression Classification

✔ Accuracy Score

✔ Confusion Matrix

✔ ROC Curve & ROC-AUC

✔ Feature Importance Analysis


🔍 Key Results & Findings

* Logistic Regression achieved good prediction accuracy.
* ROC curve showed effective classification performance.
* Features like chest pain type and cholesterol significantly affected predictions.
* Correlation heatmap helped understand feature relationships.

🚀 Conclusion

This project demonstrated how machine learning can assist in early heart disease prediction using medical data and classification algorithms.
